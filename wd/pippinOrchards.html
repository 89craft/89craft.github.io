Pippin Orchards

A prototype for an apple encyclopedia app using Adobe XD. This was to practice the process of prototyping.

My tasks for this project included:

  • Prototype wireframe,

  • Design a color scheme,

  • Create a logo,

  • Create an icon, and

  • Stylize wireframe with colors and images.
